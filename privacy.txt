2. Data Collection

We value your privacy, and we want to be transparent about the data we collect.

Data We Do NOT Collect:

3. Purpose of Data Collection

The Text Selection Search Extension does not collect or transmit any personal data. Its primary function is to allow users to search for selected text on Google. Any action taken within the extension stays within the user's browser.

4. Permissions

Our extension utilizes the activeTab permission. This means:

We only access the currently active tab when the user explicitly invokes our extension.
We do not retain any data from the active tab after the user's session.
5. Data Sharing and Third Parties

We do not share or sell any data with third parties.

6. Data Security
